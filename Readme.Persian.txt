امروز تاریخ 1403/5/17 و قرار که پیاده سازی پروژه زیر ساخت رو شروع کنیم این پروژه قرار یه پروژه آزمایشی برای یادگیری تکنولوژی های جدید باشه و اگر در آینده پروژه موفقی بود ازش استفاده بشه
برای پیاده سازی این پروژه باید یه پیش فرض ذهنی در مورد شبکه و زیر ساخت داشته باشیم تا بتونیم این مورد رو پیاده سازی کنیم که من از قبل شبکه خوندم و میدونم چی به چیه
